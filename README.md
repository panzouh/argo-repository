# Repository

Argo repository for simplified cluster management.

## Apps

- Integration :
  - ArgoCD (Argoception)
  - OLM
- Logging :
  - ECK Operator
  - Logstash or Fluentd
  - Loki
  - Promtail
- Networking :
  - Traefik
  - CertManager
- Monitoring :
  - Prometheus
  - Grafana
  - Fio exporter
- Security :
  - Vault
